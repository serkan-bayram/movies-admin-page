# movies-admin-page

> Practicing MERN skills

* Used Technologies:
    - MongoDB
    - Express
    - React
    - NodeJS

* Features:
    - Fetch movie details via TMDB API
    - 5 other options to choose correct movie
    - Add or delete movies
    - Keyboard support to find & add movies
    - Save movies within a database
